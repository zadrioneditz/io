# Installation Guide: Maxon Plugins (Serial Filler Method)

**Note: This method supports macOS (M1+ compatible) and Windows.**

## **Windows**

1. Uninstall all Maxon-related software using [**Maxon Cleaner**](https://pixeldrain.com/u/vRCSSRTu).
- After the cleanup is complete, install the version linked below.
**Important:**
Do not allow the app to update. It must remain on **version 2024.5.
[https://mx-app-blob-prod.maxon.net/mx-package-production/website/windows/maxon/maxonapp/releases/2024.5.0/Maxon_App_2024.5.0_Win.exe](https://mx-app-blob-prod.maxon.net/mx-package-production/website/windows/maxon/maxonapp/releases/2024.5.0/Maxon_App_2024.5.0_Win.exe)**
1. Sign in to your Maxon account. Create one if you do not already have it.
2. Install the plugins you want. 
Go to **Other Versions** and select any version earlier than 2023.0
(Install → Other Versions → Scroll).

<aside>

INFO :
It has to be before 2023.0, for example **Magic Bullet Looks 16** and **Universe 5**.

</aside>

**You may install the following versions:**

- **Magic Bullet:** 15 or 16
- **Trapcode:** 7 or 8
- **Universe:** 5 or 6
- **VFX Suite:** 2 or 3

<aside>

**WARNING:**
If Maxon asks you to update the app at any time, click **No**. Read carefully, because downgrading later is annoying.

</aside>

4. Download Maxon’s [**Serial Filler** tool](https://pixeldrain.com/u/iPf1ZWcG) and license your plugin(s) using the serials below. Enter each serial one by one for every Red Giant plugin you installed.

- **Magic Bullet Suite**
    
    `COBK2245921573563861`
    
    `COBK2245846563534111`
    
- **Universe**
    
    `RLPK2245855360592319`
    
- **Trapcode Suite**
    
    `TCBK2245868172939255`
    
- **VFX Suite**
    
    `VXXF1545921787095659`
    
- **Plural Eyes**
    
    `PFEK1245921362842515`
    

Now you're done, enjoy!

# **MacOS**

1. In the top right corner, select “**Exit RLM Mode**" and sign in — *(you will need to create your own account)*.
2. Uninstall **all plugins within the Maxon App** (Magic Bullet, Trapcode, Universe, etc.). *Alternative: [**Run this script in Terminal.**](https://pixeldrain.com/u/Twbz3kRx)*

<aside>

**WARNING**

**DO NOT UNINSTALL OR TOUCH CINEMA 4D.**

After Effects requires Cinema 4D to work; it **cannot** be uninstalled!

</aside>

1. Uninstall Maxon App *(this will be uninstalled already if you ran the bash file)*
2. Download [**Maxon App 2024.5.0**](https://pixeldrain.com/u/zivVikM5) & do **NOT** accept any update alerts, read carefully. Log into your Maxon account
    
    The following method does not work with newer versions of Maxon App.
    
3. Install the plugins you want, click “other versions” and choose any version before 2023.0 — *(install > other versions > scroll)*.

<aside>

**INFO**

**It has to be before 2023.0**, so Magic Bullet Looks 16 and Universe 5, for example.

</aside>

**You may install the following versions:**

- Magic Bullet 15 or 16
- Trapcode 7 or 8
- Universe 5 or 6
- VFX Suite 3 or 2

<aside>

**WARNING**

**If Maxon asks if you want to update the app at any given time, click NO.Read carefully, downgrading back will be annoying if you don't.**

</aside>

1. Download Maxon's [**Serial Filler Tool**](https://pixeldrain.com/u/4bBrCcjE) and license your plugin(s) with the serial(s) below. Enter them one by one for each Red Giant plugin you installed:
- **Magic Bullet Suite**
    
    `COBK2245921573563861`
    
    `COBK2245846563534111`
    
- **Universe**
    
    `RLPK2245855360592319`
    
- **Trapcode Suite**
    
    `TCBK2245868172939255`
    
- **VFX Suite**
    
    `VXXF1545921787095659`
    
- **Plural Eyes**
    
    `PFEK1245921362842515`
    
    Now you're done, enjoy!
